Taladro Perfecto

Velocidad variable para un máximo control con la punta de los dedos para todas las aplicaciones de perforación. Sistema de cepillo inverso para un par y potencia completos en avance y retroceso, botón de bloqueo para uso continuo.

Cómodo de usar

Diseño ergonómico de taladro percutor reversible con posiciones de agarre óptimas, peso ligero y cuerpo de plástico aislado a prueba de golpes, brocas fáciles de cambiar

Madera y metal

Admite perforaciones en madera y metal. Las brocas para metal no forman parte de los componentes incluidos con el producto. El cliente puede adquirir brocas para metal y se pueden utilizar para taladrar superficies metálicas
